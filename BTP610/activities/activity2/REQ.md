## Task: ToDo App

This is a single screen application. It should:

1. Add a task with Title and Status (The possible statuses are `due` or `done`). The status will be `due` by default. When a new task is added, it will be displayed in FlatList.
2. Add a feature to EDIT the Status of the task. The user must be able to tap a switch or checkbox or any other touchable component in the task card to easily change its status.
3. Add a feature to DELETE the selected task from the list. The user must be able to tap an icon or any other touchable component in the task card to easily delete.
4. Create a verification to allow the creation of the task ONLY when there is a title. Do not overcomplicate it. This could be done, for example, by simply disabling the `ADD TASK` button when the input field is empty.
5. Implement UI/UX with the use of appropriate components to accept input from the user and display proper output.
